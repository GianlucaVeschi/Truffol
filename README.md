# Truffol
Mobile Android App to connect truffles retailers in Italy with customers in Germany and the rest of the EU.

## Tech Stack
- Kotlin
- MVVM
- Jetpack Compose
- Dagger Hilt
- Coroutines
- Retrofit / OkHttp
- Room DB
- Junit5
- Firebase SDK
- Glide
- MockWebServer
- And more...
